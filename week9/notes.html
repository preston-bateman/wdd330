Chapter 9

Browser Object Model

  • BOM is collection of properties and methods contain info about browser and computer screen.
  • These properties are available through window object. BOM only makes sense when JS is run in browser.
  • Global variables are properties of the global object which is the window in the browser.
  • alert() confirm() and prompt() are not part of JS but are part of the window object.
  • .prompt() takes a dialog as argument and then returns the response from the user.

Browser Information

  • window.navigator object has info on the browser and the operating system
  • window.location contains info on the URL of the current page. the .reload() property of location can force a reload.

Browser History

  • window.history contains info on previous visited pages in the current browser session.
  • history.go() with a number as the argument will go to previous and forward pages

Conrolling Windows

  • You can open a new window with the window.open() method. The first parameter is the website to be opened, window title as second, and list of attributes as the third
  • .close and .moveTo can close a window and move it on the screen respectively

Screen Info

  • window.screen has the height and width properties that can be changed. .availHeight and .availWidth tells you the available screen real estate minus any menus.

Document Object

  • window object has the document object.
  • You can use the .write to write to the screen.
  • cookies are limited to 4kb of data
  • Cookies are in the form of a text file that have name value pairs.
  • You create a cookie by adding it to the document.cookie property.
  • You read cookies with the split method that breaks the string into an array that you loop through.
  • You make cookies persistent accross sessions with the expires=date at the end of the cookie string.
  • You delete cookies by setting the expiration date to the past.

Timeout Functions

  • .setTimeout() method accepts callback function as first parameter and number of milliseconds as the second.
  • .clearTimeout() will kill any outstanding timeouts

setInterval

  • Similar to the setTimeout() method this method will call a function every set amount of time.
  • .clearInterval is used to stop the method.

Chapter 14

Data attribute

  • data- is a way of embedding data in a page that the browser ignores.
  • Naming Rules: start with "data-", contain only lowercase letters numbers hyphens dots colons underscores, include optional string value.

HTML 5 APIs

  • HTML 5 introduced Local storage as well as session storage.
  • Storage is limited to about 5gb.
  • Methods include: getItem, setItem, remove, delete, and clear.
  • Only strings can be used to store items. You can use the JSON.stringify method to store objects easily. They are then put in an array.

Drawing With Canvas

  • canvas element was introduced to allow drawing in HTMl using javascript.
  • Methods: fillStyle, strokeStyle, lineWidth, fillRect, strokeRect, beginPath, moveTo, endPath, lineTo, stroke
