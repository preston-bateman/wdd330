N2N Chapter 5: Objects

Object Literals

  • Objects used to keep related information and functionality together in a single entity.
  • Object literal is an object created by wrapping properties (key value pairs of primitive data) and methods (key value pairs of functions) in curly braces {}.
  • If a property name has space in it, it must be put in ' '.
  • All Object properties and methods are mutable (changeable) while running a program, even if it is defined using const

Creating Objects

  • Example: const objectName = {};
  • Using Constructor Function: const objectName = new Object(); not recommended because too much typing
  • ES6 provided a shorthand for property names if the key is the same as a variable the property is assigned to:
    • const name = "some name";
      const otherproperty = "example text";

      const objectName = {name, otherproperty};
  • Access properties using dot notation example: objectName.objectProperty
  • Access properties using bracket notation example: objectName['objectProperty']
  • Values of a property can be a JavaScript expression, like a ternary operator or Symbol
  • Using Symbols for the key is useful as they all have a unique value and that helps to avoid naming clashes.

Calling Methods

  • Use dot notation example: objectName.objectMethod()

Checking if Properties or Methods Exist

  • Use the in operator example: 'propertyName' in objectName;
  • Also can use: objectName.hasOwnProperty('propertyName');
    • This will only return true if the object has the property but won't return it if it is an inherited property. IN will return the true even with an inherited property

Finding All Properties of Object

  • for(const key in superman) {
     console.log(key + ": " + superman[key]);
    }
  • Create an array with all of an objects keys: object.keys()
  • Create an array with all of the object values: object.values()
  • For key value pairs array: object.entries()

Nested Objects

  • Objects can be nested in one another. Just add another dot notation to access the propertys of the nested object
  • Objects are assigned by reference. If you assign a variable an existing object, then it will point to the same info in the memory. Changing the variable will change the object.

Object as Parameter to Functions

  • This is useful when you are passing a large number of parameters to a function

Object This

  • This is used to refer to properties withing the object.

Namespacing

  • You can use objects as a store of functions so that you don't run into naming collisions. The object functions like a library of functions for you to draw from.

JSON

  • Used commonly for data serialization(storing objects in a series of bytes that can be converted back to an object) and configuration.
  • JSON is a string notation of an object
  • JSON special rules:
    • 1. Property names must be double-quoted ""
    • 2. Permitted values are double-quoted strings, numbers, true, false, null, arrays, and objects
    • 3. Functions are not permitted values
  • The .parse() method takes a JSON string and returns a JS object. Example: JSON.parse(jsonvariable)
  • The JSON.stringify(objectName) method converts an Object into JSON data format.
  • Methods in a stringified object are just ignored and not included.
  • You can also use JSON.stringify(objectName, null, " ") to create a JSON that will add a space argument between key value pairs

The Math Object

  • Math object has 8 properties that are a selection of commonly used math constants:
    • Math.PI is pi
    • Math.SQURT2 is the square root of 2
    • Math.SQURT1_2 is the recipricol of the square root of 2
    • Math.E is Euler's Constant
    • Math.LN2 is the natural logarithm of 2
    • Math.LN10 is the natural logarithm of 10
    • Math.LOG2E is Log base 2 of Euler's constant
    • Math.LOG10E is Log base 10 of Euler's constant
  • Math Methods:
    • Math.abs() method returns the absolute value of a number. So if the number is positive, it will remain the same, and if it's negative, it will become positive
    • The Math.ceil() method will round a number up to the next integer, or remain the same if it is already an integer
    • The Math.floor() method will round a number down to the next integer, or remain the same if it is already an integer
    • The Math.round() method will round a number to the nearest integer
    • Math.trunc() method that returns the integer-part of a number that is, it gets truncated at the decimal point
    • The Math.exp() method will raise a number to the power of Euler's constant
    • The Math.pow(1st arg, 2nd arg) method will raise any number (the first argument) to the power of another number (the second argument)
    • The Math.sqrt() method returns the positive square root of a number
    • The Math.cbrt() method was introduced in ES6, which returns the cube root of numbers
    • The Math.hypot() returns the square root of the sum of the squares of all its arguments.
    • The Math.max() method returns the maximum number from its arguments
    • And the Math.min() method unsurprisingly returns the minimum number from the given arguments
    • Math.random() creates a random generated number between 0 and 1
    • You can make that a random integer between 0 and a specified number like this: Math.floor(specifiedNumber * Math.random());
  • The Date Object: const today = new Date(). Date parameters are: new Date(year, month, day, hour, minutes, seconds, milliseconds)
  • Use the .toString() method returns the date in full length format.
  • An alternative is to use a timestamp, which is a single integer argument that represents the number of milliseconds since the Epoch (1st January 1970): new Date(1508367600000)
  • Date Obj Getter Methods
    • getTime(), getTimezoneOffset(), getYear() don't have UTC equivalents
    • getDay(), getUTCDay() find the day of the week and returns 0-6
    • getDate(), getUTCDate() return the day of the month. These start counting at 1 not 0
    • getMonth(), getUTCMonth() return the month of the year with a 0-11
    • getFullYear(), getUTCFullYear() returns the year of the date
    • getHours(), getMinutes(), getSeconds(), getMilliseconds()
    • getTime() returns the number of milliseconds from the epoch Jan 1 1970
  • The Getter methods have equivilent setter methods. Example: .setMonth(10)

The Document Object Model

DOM

  • DOM represents the HTML document as a network of connected nodes that has a tree like structure.
  • The DOM treats everything on a page as a node. Tags function as the root node with all it's parts as children of the node.

Getting Elements

  • Methods that let us access a page element return a node object or node list(an arra-like object). These can be assigned to a variable and modified or viewed.
  • variableHoldNodeObj.nodeType will show us what type of node it is:
    • 1 = element
    • 2 = attribute
    • 3 = text
    • 8 = comment
    • 9 = body
  • variableObj.nodeName property to find the name of the element

Legacy DOM Shortcut Methods

  • These are left over from DOM level 0:
    • Document.body returns the body element of a web page, as we saw in the previous example.
    • Document.images returns a node list of all the images contained in the document.
    • Document.links returns a node list of all the "a" elements and "area" elements that have an href attribute.
    • Document.anchors returns a node list of all the "a" elements that have a name attribute.
    • Document.forms returns a node list of all the forms in the document

Get Element By ID

  • Document.getElementById() retrieves the element with the unique ID specified
  • Only use an ID once in an HTML document or it can cause odd issues in the code.

Get Element By Tag Name

  • document.getElementsByTagName() will return a live node list of all the elements with the tag name. You can sort through them with []

Getting Elements by their class

  • document.getElementsByClassName() gives a live node list

Query Selector

  • document.querySelector() allows you to get the first element in a document that matches the CSS Selector given as an argument.
  • document.querySelectorAll() functions the same but returns a node list that return all elements that match the CSS query Selector.
  • This methodology is very powerful as it allows you to use CSS pseudo-selectors
  • querySelector() can be called on any element, not just the document element. This includes element objecst that are stored in a variable.

Navigating the DOM Tree

  • .childNodes property will show you all of the children nodes. This returns all children and not just the element children of the node
  • .children will return just the element children nodes of a node
  • There is also .firstChild and .lastChild
  • .parentNode will return the parent node of an element
  • .nextSibling returns the next adjacent node of the same parent. There is also a .previousSibling
  • .textContent and .innerText access the text of an Element

Getting and Setting Attributes

  • .getAttribute('') method returns the value of the attribute of the provided argument
  • .setAttribute('attributeName','attributeValue') changes the value of the selected attribute. It takes two arguments.
  • .setAttribute() can also add an attribute that isn't already there.
  • .className is a property that allows you to view and modify the class of an element.
  • .classList lets you see all the classes of an element. .classList.add() will add the argument to the elements class list.
  • .classList.remove() will remove from the class list
  • .classList.toggle() will add if the class doesn't already exists or remove it if it does

Creating Dynamic Markup

  • createElement() method allows you to create a new tag.
  • document.createTextNode('textString') method allows you to create a text node.
  • .appendChild() method lets you add another node as the last child of a parent node
  • It is better to create a function to add elements.
  • function createElement (tag, text) {
    const el = document.createElement(tag);
    el.textContent = text;
    return el
    }
  • .insertBefore(addElement, existingElement) method will add an element before an existing element.
  • .appendChild() and .insertBefore() can also be used to move an element in the web page
  • .removeChild() will remove an element. This is called on the parent node and the parameter is the node to be removed.
  • .replaceChild(newElement, oldElement) functions similarly but just replaces the node with another
